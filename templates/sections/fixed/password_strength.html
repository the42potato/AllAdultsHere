How it Works
  • Entropy is calculated from a set of possible characters. It is good at estimating brute-force resistance, but does not account for dictionary words and other predictible patterns.
  • Calculated points has a similar issues to entropy as they do not account for predictible patterns. Points are gained from:
    • Containing a lowercase letter (+1)
    • Containing an uppercase letter (+1)
    • Containing a digit (+1)
    • Containing a special character (+2)
    • Containing eight or more characters (+1)
    • Containing ten or more characters (+1)
  • The pattern score utilizes zxcvbn, which returns a 0-4 rating based on how close the password matches various patterns.
Rating Calculation
  • Using the last three components, we can then create a formula for an overall rating:
    • Each component has a weight (entropy 75%, points 100%, and pattern matching 200%) which determines how much they influence the final calculation.
    • each component is multiplied by its weight, summed, and reduced to fit within the range 0-4, leaving us with five ranks that can tell you roughly how secure your password is.
